🚀 Gemini Clone

A modern AI chatbot web application inspired by Google Gemini, built using React.js and powered by the Gemini API.
This project demonstrates how to build a real-time conversational AI interface with a clean UI and API integration.

📌 Overview

This application allows users to interact with an AI chatbot in a conversational format.
It mimics the behavior of Google Gemini by providing intelligent, real-time responses.

Perfect for:

Learning API integration
Building chat-based UIs
Practicing React development

✨ Features:
💬 Interactive Chat Interface
⚡ Real-time AI Responses
⌨️ Typing Animation Effect
🎨 Clean & Responsive UI
🔗 Gemini API Integration

🛠️ Tech Stack:
Frontend: React.js, JavaScript, HTML, CSS
API: Google Gemini API
Build Tool: Vite / NPM
⚙️ Installation & Setup

Follow these steps to run locally:

# 1. Clone the repository
git clone https://github.com/Aayush-0409/Gemini-Clone.git

# 2. Navigate into the project
cd Gemini-Clone

# 3. Install dependencies
npm install

# 4. Add your Gemini API key
# Create a .env file and add:
VITE_GEMINI_API_KEY=your_api_key_here

# 5. Run the app
npm run dev
🌐 Usage
Open browser and go to:
http://localhost:5173
Enter your prompt in the chat box
Get AI-generated responses instantly
